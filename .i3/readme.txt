Requires nitrogen for backgrounds, redshift for color temperature

Droid sans as font

xfce4-terminal as term emulator- config is gui (easy)

dmenu as launcher

lxappearance and paper theme for gtk themeing

#Setup

run `i3init.sh`

change xfce4-terminal settings to hide menubar, use droid sans mono and solarized dark

use lxappearance to change system font to droid sans, and use paper for gtk/icon themes

Change chrome fonts to use droid
